# Changelog

## 0.1.0 — Initial Fork

Forked from [gstack](https://github.com/garrytan/gstack) v0.13.3.0.

### What changed

- **New ethos:** Revenue Before Perfection, Ship Then Iterate, Don't Build What You Can Buy, One Person One Repo, Search Before Building.
- **Kept all 29 skills** from gstack — solo founders need every role even more than teams do (you ARE the whole team).
- **Renamed** all gstack references to pstack.
- **New README** focused on bootstrappers, indie hackers, solo founders.

### Coming soon

- `/validate` — validate willingness to pay before building
- `/launch` — landing page → waitlist → deploy pipeline
- `/monetize` — Stripe/LemonSqueezy setup in one command
- Simplified ship workflow (push to main, no PRs by default)
